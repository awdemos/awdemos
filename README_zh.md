## 🚀 推介

**我完全用 Rust 构建了完整的堆栈——从内核到 UI——完全从头开始。**

一个操作系统。文本编辑器。人工智能编码助手。区块链。借款检查员。代理协调者。神经网络。编程语言。合成生命模拟器。

全部用 Rust 编写。全部免费。全部开源。

> **个人延伸目标：** 构建我依赖的*每一个*工具——我使用的任何东西都应该是我自己无法编写的。堆栈的完全所有权。

我免费赠送所有这些内容，以向所有直接或间接影响我的已知和未知的开源黑客致敬。

## 🧭 我对建筑的看法

> **可重复。可观察到的。可组合。**

我根据第一原则设计系统。能把一件事做好并干净利落地组合的小工具——Unix 方式，但适用于人工智能时代。

- **我不信任黑匣子。** 如果我无法审核它，我就不会发货。
- **与 YAML 相比，我更喜欢代码。** 如果它可以是代码，它就应该是代码。
- **我相信人工智能主权。** 你拥有的模型、你控制的数据、你可以审计的基础设施。
- **我重视内存安全。** 不是因为 Rust 告诉我这样做，而是因为在凌晨 3 点调试段错误是浪费生命。
- **我是久经考验的真实原则和未被充分重视的技术的忠实粉丝。** 我可以同样流畅地处理从 PDP-11 到 FreeBSD 和后 Linux 内核的所有内容。

我的成长岁月是在 **Slashdot** 度过的，并深入**开源文化**——透明、协作和公共建设的精神仍然是我今天的工作方式。

这项工作的大部分工作都基于一个想法：**智能的未来是统一的，而不是支离破碎的。**

我正在探索构建 **AI Pangaea** 意味着什么——将分布式智能岛屿融合成一个更有凝聚力、更安全、更有能力的连续体：
