## 🧭 Hur jag tänker på att bygga

> **Reproducerbar. Märkbar. Komponerbar.**

Jag designar system från första principer. Små verktyg som gör en sak bra och komponerar rent — på Unix-sättet, men för AI-eran.

- **Jag litar inte på svarta lådor.** Om jag inte kan granska den skickar jag den inte.
- **Jag föredrar kod framför YAML.** Om det kan vara kod ska det vara kod.
- **Jag tror på AI-suveränitet.** Modeller du äger, data du kontrollerar, infrastruktur du kan granska.
- **Jag värdesätter minnessäkerhet.** Inte för att Rust sa åt mig att göra det, utan för att felsökning av fel vid 03:00 är ett slöseri med ett liv.
- **Jag är ett stort fan av beprövade principer och underskattade teknologier.** Jag kan arbeta med allt från en PDP-11 till FreeBSD och post-Linux-kärnor med lika flytande egenskaper.

Mina uppväxtår spenderades på **Slashdot** och djupt inne i **kulturen med öppen källkod** – att etiken om transparens, samarbete och byggnad i offentligheten fortfarande är hur jag arbetar idag.
