## 🚀 ピッチ

**私はカーネルから UI までのフルスタックを完全に Rust で、完全にゼロから構築します。**

オペレーティング システム。テキストエディター。 AIコーディングアシスタント。ブロックチェーン。借用チェッカー。エージェント オーケストレーター。ニューラルネットワーク。プログラミング言語。人工生命のシミュレーター。

すべてRustで書かれています。すべて無料です。すべてオープンソース。

> **個人的なストレッチ目標:** 私が依存している *すべての* ツールを構築する — 私が使用するものは、自分で作成できなかったものであってはなりません。スタックの完全な所有権。

私に直接的または間接的に影響を与えた、既知および無名のオープンソース ハッカー全員への感謝の気持ちを込めて、これらすべてを無料で公開します。

## 🧭 建築についての私の考え方

> **再現可能。観察可能。コンポーザブル。**

私は第一原理からシステムを設計します。 1 つのことをうまく実行し、きれいに構成する小さなツール — Unix のやり方ですが、AI 時代向けです。

- **ブラック ボックスは信用しません。** 監査できない場合は出荷しません。
- **YAML よりもコードの方が好きです。** コードにできるのであれば、コードであるべきです。
- **私は AI の主権を信じています。** あなたが所有するモデル、あなたが管理するデータ、あなたが監査できるインフラストラクチャ。
- **私はメモリの安全性を重視しています。** Rust からそうするように言われたからではなく、午前 3 時にセグメンテーション違反をデバッグするのは人生の無駄だからです。
- **私は実証済みの真の原則と過小評価されているテクノロジーの大ファンです。** PDP-11 から FreeBSD やポスト Linux カーネルまで、あらゆるものに同等の流暢さで取り組むことができます。

私の形成期は **Slashdot** と **オープンソース文化**の奥深くに費やされました。その透明性、コラボレーション、公共の場での構築という精神が、今でも私の仕事の仕方となっています。

## 🌌 全体像: AI Pangea

この研究の多くは、**インテリジェンスの未来は断片化ではなく統合される**という 1 つのアイデアに基づいています。

私は、**AI Pangaea** を構築することが何を意味するのか、つまり、分散したインテリジェンスの島々を、より結束性が高く、安全で、有能な連続体に収束させることが何を意味するのかを模索しています。

- 永続的な記憶と再帰的な自己改善を備えた **超知能エンティティ**
- **あらゆる層のセキュリティ** — サンドボックス、ガードレール、コンセンサス検証、機密実行
- **セッション、デバイス、インテリジェンス アイランドにわたる **メモリと継続性**
